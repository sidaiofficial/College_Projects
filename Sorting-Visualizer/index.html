Description

QuickSort is a Divide and Conquer algorithm. It picks an element as a pivot and partitions the given array around the picked pivot..

Complexity:
Average Complexity:   O(nlogn)
Best Case:   O(nlogn)
Worst Case:   O(n2)
Space Complexity:   O(logn)
